## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Bikiran007/Thumblify--Ai-Thumbnail-Generator.git
cd Thumblify--Ai-Thumbnail-Generator
```

### 2. Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### 3. Setup environment variables

Create a `.env` file inside the `server` folder:

```env
API_KEY=your_api_key_here
PORT=5000
```

### 4. Run the application

```bash
# Start backend
cd server
npm start

# Start frontend (in another terminal)
cd client
npm start
```

## 🧠 How It Works

1. User enters a prompt for thumbnail
2. Frontend sends request to backend
3. Backend calls AI API
4. AI generates thumbnail
5. Image is returned and displayed to user
